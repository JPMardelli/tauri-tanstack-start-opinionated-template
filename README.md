# Tauri + Tanstack Start

This project is an attempt to create a reusable starter for a Tauri Native app using Tanstack Start as its frontend

On the frontend side, it will have several opinionated inclusions:

- `typescript` - language
- `bun` - runtime and package manager
- `rsbuild` - build tool
- `rstest` - testing tool
- `biome` - linter
- `vanilla-extract` - CSS-in-JS styling
